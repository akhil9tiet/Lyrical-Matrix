View your app in AI Studio: https://ai.studio/apps/drive/1FSdln3OcZgxbR9wGnHlDBbx6KFgi5yi_

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
